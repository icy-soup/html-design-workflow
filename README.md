# html-design-workflow

用中文一句话，让 Claude 帮你生成高质量 HTML 页面、技术文档、复习资料或落地页。

## 功能

- **分类路由**：复习资料/考试备考 → 默认分析布局（卡片式章节+侧边栏TOC）；技术文档 → 左侧导航+内容区；幻灯片 → 全屏翻页；落地页 → 品牌聚焦
- **默认分析布局**：卡片式章节、彩色章节标记、公式卡片、定理框、提示框、易错警示、记忆口诀、频率标签、样式化表格
- **侧边栏目录**：自动从 h2/h3 生成 TOC，滚动高亮当前章节，移动端汉堡菜单
- **点击显示**：答案/证明/记忆内容可设置点击展开（开卷考场景自动跳过）
- **KaTeX 数学公式**：支持 LaTeX 数学渲染，自动处理 `<` 转义与 CDN 优化（cloudflare，国内友好）
- **风格学习**：给 Claude 看一个 HTML 文件，它能提取色调/字体/布局并复用到下一个需求
- **零 ASCII 图**：所有流程图/图表用 HTML/CSS/SVG 渲染，禁止 `┌┐└┘├┤─│` 等字符
- **规范交付**：内置交付清单，含 KaTeX 校验、侧边栏功能验证等

## 目录结构

```
├── SKILL.md                          # 主 skill 定义
├── references/
│   ├── diagram-components.md         # 图组件详细参考
│   ├── analysis-layout.md            # 默认分析布局样式指南
│   ├── sidebar-toc.md                # 侧边栏目录实现
│   └── click-to-reveal.md            # 点击显示交互实现
└── styleref/                         # 已学习的风格记录
```

## 触发条件

当你说「做 HTML / 做网页 / 做复习资料 / 做知识整理 / 做文档 / 做手册 / 做幻灯片 / 做 PPT / 做页面 / 做组件 / 前端设计 / UI 设计 / 学习这个风格 / 参考这个页面 / 提取风格」时自动触发。

也触发于：「公式不显示 / 公式乱码 / 公式渲染 / KaTeX / $$ 显示问题 / 数学符号乱码 / 数学公式 / 复习资料HTML / 期末复习 / 课件HTML」等与数学公式渲染相关的问题。
